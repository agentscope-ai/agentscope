项目名：agentscope_demo
目标：构建一个 agentscope 中 agent cli demo
技术栈：Python(uv) + agentscope library
代码规范：pre-commit检查
要求：
- 使用 uv 的 python 环境
- 实现一个 cli 版 agent 和用户交替交流的样例，当用户在 terminal 中输入exit的时候退出
- 需要支持 human-in-the-loop 的 feature，即让用户在cli上确认工具的执行
参考资源：
- agentscope 文档：https://docs.agentscope.io
- agentscope repo: https://github.com/agentscope-ai/agentscope
