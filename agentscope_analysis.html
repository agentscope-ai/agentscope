What is it

The framework that gets out
of your model's way

▲

The Problem

Most agent frameworks constrain LLMs with rigid prompts, hard-coded orchestration, and opinionated pipelines. Every hardcoded rule is technical debt as models improve.

◎

The Philosophy

AgentScope leverages the model's own reasoning and tool-use abilities. You provide structure at the edges — memory, deployment, observability — and let the LLM think.

◈

The Audience

Built for engineers who need to ship production agents, not demos. Supports local, serverless, and Kubernetes deployment with OpenTelemetry observability out of the box.

◐

The Differentiator

Unique agentic RL support: train your agents end-to-end with reinforcement learning using the Trinity-RFT library. Agents learn to be better agents — not just prompted to be.

Hello, AgentScope

A ReAct agent with tools, memory, and streaming — in under 30 lines. Works with any OpenAI-compatible API, DashScope, Anthropic, and more.

The Toolkit accepts any callable — including MCP tools unwrapped as local functions.

from agentscope.agent import ReActAgent, UserAgent from agentscope.model import DashScopeChatModel from agentscope.memory import InMemoryMemory from agentscope.tool import Toolkit, execute_python_code agent = ReActAgent( name="Friday", sys_prompt="You're a helpful assistant.", model=DashScopeChatModel( model_name="qwen-max", api_key=os.environ["API_KEY"], stream=True, ), memory=InMemoryMemory(), toolkit=Toolkit(), ) # Run conversation loop async def main(): msg = None while True: msg = await agent(msg) msg = await user(msg)

Get started

From zero to
running agent in 5 min

01 / Install

Add to your project

Requires Python 3.10+. Works with pip or uv.

pip install agentscope
02 / Configure

Set your API key

Works with DashScope, OpenAI, Anthropic, or any OpenAI-compatible endpoint.

export DASHSCOPE_API_KEY=...
03 / Build

Create your agent

ReActAgent is the default — gives your agent tools, memory, and streaming out of the box.

agent = ReActAgent(...)
