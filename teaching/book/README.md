# 《Java开发者学Python Agent开发——基于AgentScope》

> **致敬经典**：本书借鉴《网络是怎样连接的》(户根勤 著) 的写作风格

---

## 🔍 探索之旅：从"你好"到"我是Agent"

```
用户输入"你好" → Agent → Model → Tool/Memory → Response → 用户看到回复
```

**每一站都追踪数据在其中流动的过程**，不是孤立地讲组件，而是讲它们如何协作。

## ✨ 书籍特色

1. **追踪式学习** - 追踪数据在系统中的完整旅程
2. **图解优先** - 每一步都用追踪图说明
3. **Java锚定** - 用Java系统类比理解Python概念
4. **术语说人话** - 简单易懂地解释每个术语
5. **先跑起来** - 代码完整可运行
6. **项目驱动** - 每站有思考题，项目有完整代码
