# AgentScope 学习指南

欢迎来到 AgentScope 学习中心！本指南专为有 Java 背景的开发者设计，帮助你快速掌握这个 Python 多智能体框架。

## 前置知识

- 了解 Java 面向对象编程
- 熟悉设计模式（工厂、策略、模板方法）
- 最好有 REST API 或微服务开发经验

## 你能学到什么

1. **多智能体系统基础** - 如何编排多个 AI 智能体协同工作
2. **LLM 应用开发** - 集成 OpenAI、Anthropic 等大模型
3. **工具调用** - 让 AI 智能体调用外部工具
4. **记忆管理** - 实现短期/长期记忆
5. **RAG 检索增强** - 知识库问答
6. **实时语音交互** - 构建语音对话应用
